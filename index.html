【一】高德地图 市区数据监控 点击进入追踪

1.运行

* $ git clone 地址
* npm install 
* $ gulp;

2.项目说明（吐槽）

  • 此项目是 百度版--市区监控--实时追踪 的复制模块（高德版）。具体业务逻辑和百度这个模块的一模一样。以下就是吐槽
  • 很多地方和baidu模式极其极其极其相似，相似到我都觉得高德地图的API的名字都是照搬百度的API。
